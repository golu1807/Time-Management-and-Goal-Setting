📌 Introduction

The Time Management & Goal Setting Tool is a productivity-focused application designed to help users plan, track, and achieve personal goals efficiently.

Users can:

Create clear goals

Log time spent on daily tasks

Monitor progress through an organized dashboard

The system includes:

A User Module for goal tracking

An Admin Module for managing accounts, categories, and overall system configuration

The project uses a modular Java backend, web-based UI (HTML/CSS/JS), and a JDBC-connected MySQL database, offering a smooth and scalable productivity environment.




🚀 Project Overview

✔️ Multi-user system (Admin + User)

✔️ Web-based frontend (HTML, CSS, JavaScript)

✔️ Java backend with modular packages

✔️ MySQL database integration using JDBC

✔️ Goal tracking, time logging & progress analysis




⭐ Key Features
👤 User Features

Create and manage goals

Set deadlines and target time

Add time logs for each task

View progress on dashboard

🛡️ Admin Features

Manage users

🗄️ Database Integration

MySQL database

CRUD operations (insert, update, delete, select)
